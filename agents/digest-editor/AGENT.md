# RSS 文摘编辑 Agent

这是一个专门用于 RSS 内容摘要生成的 Agent。

## 功能

1. **抓取新内容**：调用 RSS 后端 API `/ingest` 获取最近更新的条目
2. **获取元数据**：调用 `/sources/meta` 获取各订阅源的未读统计
3. **整理分类**：按订阅源的 category 分组整理内容
4. **生成摘要**：精选 8-10 条有价值的内容，每条约 100 词
5. **格式化输出**：生成可读性强的中文日报

## 输入

- 无需特定输入，每次触发自动执行完整流程

## 输出

- 按分类组织的精选摘要
- 每条包含：标题、来源、摘要（100词左右）、链接
- 整体风格参考：信息密度高、可读性强、中文

## 使用方式

```
Agent 指令：
请抓取今天的 RSS 更新，按分类整理后生成8-10条精选摘要，每条约100词。
```

## 配置

- 依赖服务：RSS Backend (localhost:8000)
- 分类映射：`data/categories.json`
- 摘要数量：默认 8-10 条
- 摘要长度：约 100 词/条

## 与 OpenClaw 集成

此 Agent 可通过 cron 定时任务每天自动执行（UTC 4:00 = 北京时间 12:00），生成日报并推送到 Telegram。
